Theming, Open Source, Typescript - Code Package Update

Release 4
Released On: October 22, 2019
New in the Release
React Core Components are now Open Source
We have decided to publish our core code to github and make the library available via NPM. There are several reasons, such as change and improvement requests that are coming from you guys. We believe that this will also ease your development as well as it keeps your code up to date! https://github.com/march08/dashboard-ui-kit‍
Dark Theme or any theme!
With refined CSS, it’s super simple to create a new theme for your dashboard within few CSS lines of code! We’ve already prepared a dark theme for you, the rest is up to your creativity!
Typescript
One thing that was missing is support for typescript. Now the new core code is written in TS natively! Unfortunately we’ve discontinued support for Flow.
Improved
CSS is based on Bootstrap classes - The previous version wasn’t very friendly for CSS overrides. We’ve decided to use bootstrap classes instead, though Bootstrap CSS file is not included in the React version and is not required!
Bootstrap Support - If your application relies on Bootstrap then good news! Pre-rendered HTML files are ready to be used with Bootstrap CSS/JS as well!
How to get all the new files?
Our project files and releases are delivered to you via Gumroad. After the purchase you should receive a receipt in your email. But to be sure you always get the latest files here's a video how to get them!
